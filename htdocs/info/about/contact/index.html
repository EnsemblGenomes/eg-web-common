Contact Ensembl Genomes

Ensembl Genomes has been developed using code originally developed by the Ensembl project. Whereas Ensembl concentrates on vertebrate species, Ensembl Genomes focuses on non-vertebrate species. There are two mailing lists on Ensembl, and one specifically for Ensembl Genomes.

Correspondence on any aspect of Ensembl or Ensembl Genomes is encouraged.

Help

Online Help

For general questions about Ensembl Genomes, please check the online documentation. If you cannot find an answer, or if you have a problem with the website, you can contact our help desk.

Mailing Lists

The Ensembl and Ensembl Genomes mailing lists are ideal for programmers, code and installation discussions.
