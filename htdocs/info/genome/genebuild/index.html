Gene annotation in Ensembl Genomes

Main genebuild

Data in Ensembl Genomes comes from a variety of sources; in some cases, genes are built by collaborators; in some cases, we are actively involved ourselves in an Ensembl-style genebuild process; in other cases, databases are built from assembling information already available in the public archives

See the homepage for each species or collection for more information about how genes in this data set have been built.
