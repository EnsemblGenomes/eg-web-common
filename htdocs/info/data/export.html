Export via website

Many of the pages displaying Ensembl genomic data offer an export option, suitable for small amounts of data.

This is the ideal option if you want a short sequence as a FASTA file, or a GFF file of a few features in a region. Simply find your desired feature or region, click on one of the "Export..." links in the left-hand menu, and select your output options.

We respectfully request that you do not script against the export pages on the Ensembl website, as this degrades the service for other web visitors. The public MySQL server is provided specifically for this purpose. Thank you.

If you wish to extract multiple features or regions, we recommend using the Perl API if possible.
